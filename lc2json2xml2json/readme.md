# lc2json2xml2json.exe

## usage

the following required arguments are:
  --input <INPUT>
  --xslt <XSLT>
  --output <OUTPUT>

Usage: lc2json2xml2json.exe --input <INPUT> --xslt <XSLT> --output <OUTPUT>

For more information, try '--help'.
