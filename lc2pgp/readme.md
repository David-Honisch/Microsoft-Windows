LC2PGPEncryption v.1.0
A CLI tool for PGP key generation and encryption/decryption

Usage: lc2pgp.exe [OPTIONS]

Options:
      --generate           Generate a new PGP key pair
      --message <message>  Message to encrypt
      --name <name>        Name associated with the key
      --email <email>      Email associated with the key
      --output <output>    Output file for the key
  -h, --help               Print help
  -V, --version            Print version
