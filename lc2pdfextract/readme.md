# lc2pdfextract.exe
(for Microsoft Windows)

An application that extracts all data from pdf. It saves all images and extracts the text to the console.

## Usage
the following required arguments are required:
  --file <FILE>

Usage: lc2pdfextract.exe --file <FILE>

For more information, try '--help'.

### Example

lc2pdfextract.exe --file test.pdf
